# 💬 Simple Chat App

Aplicación básica de **mensajería** desarrollada en **Kotlin** para aprender cómo implementar la comunicación entre diferentes dispositivos.

Este proyecto fue creado como ejercicio para entender el funcionamiento básico de un **chat en tiempo real**.

---

# 📌 Descripción

Simple Chat App es una aplicación sencilla que permite enviar y recibir **mensajes entre usuarios desde distintos dispositivos**.

El objetivo del proyecto es practicar la lógica necesaria para implementar un sistema de mensajería y comprender cómo se comunican diferentes clientes dentro de una aplicación.

---

# ✨ Funcionalidades

- 💬 Enviar mensajes
- 📩 Recibir mensajes en tiempo real
- 📱 Comunicación entre diferentes dispositivos
- 🧾 Historial básico de mensajes durante la sesión
